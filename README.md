## 📂 Projects

### 🎮 Gesture Controlled Subway Surfers

A computer vision system that allows players to control **Subway Surfers using hand gestures** detected through a webcam using **OpenCV and MediaPipe**.

### 🏃 Gesture Controlled Temple Run

A gesture recognition system that enables players to **control Temple Run using real-time hand movements**.

### 🖱 Virtual Mouse Using Hand Gestures

A computer vision application that allows users to **control the computer cursor using hand gestures detected by a webcam**.

### 🔢 Finger Based Math Calculator

A gesture-based calculator that performs **arithmetic operations using finger recognition**.

### 😊 Face Emotion Detection

A machine learning system that detects **human emotions from facial expressions using real-time webcam input**.

### 🧩 Sudoku Solver using CNN

A computer vision system that **detects Sudoku puzzles, recognizes digits using a CNN model, and automatically solves the puzzle**.
